## BUILDING

Build the cart by running:

```shell
zig build
```

Then run it with:

```shell
w4 run zig-out/bin/cart.wasm
```

For more info about setting up WASM-4, see the [quickstart guide](https://wasm4.org/docs/getting-started/setup?code-lang=zig#quickstart).
